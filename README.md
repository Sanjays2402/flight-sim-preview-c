# Sky Courier

Single-file browser flight sim built with Three.js, focused on gameplay: cargo deliveries between four procedurally placed airports, time-attack ring-gate courses, mid-air speed-boost rings, fuel and hull-damage management, dynamic wind affecting flight, and a corner mini-map showing airports and the active destination. Keyboard controls: W/S throttle, arrows pitch & roll, A/D yaw, Space brakes, R reset, M new mission. Just open `index.html` (or serve with `python3 -m http.server`) — no build step.
